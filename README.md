# Transparent Background Removal (Python)

Simple image background removal using the `transparent-background` library with configurable settings and high-precision performance measurement.

---

## 🚀 Features

- Image background removal
- Multiple output types (rgba, blur, white, green screen, etc.)
- Configurable via `config.json`
- High-precision processing time measurement using `time.perf_counter()`
- Easy-to-use pipeline
